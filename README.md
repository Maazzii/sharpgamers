# #GAMERS

## 프로젝트 개요
<br>

### 기획 의도

1. 배경
    
    > PC 온라인 게임과 달리 콘솔 및 패키지 게임은 정보를 공유할 곳이나 함께 게임을 즐길 사람을 찾기가 힘들다.
    > 

1. 목적
    
    > 게임 별 게시판을 제공해 정보를 공유<br>
    소모임을 누구나 개설해서 친목 활동 가능<br>
    회원간 1:1 메시지를 통해 소통 가능
    >
